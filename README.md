## 📌 Sobre

Este projeto foi desenvolvido como parte das atividades do curso **Alura – LangGraph: Orquestrando agentes e multiagentes**. O objetivo do curso foi aprofundar o entendimento sobre arquiteturas baseadas em agentes utilizando grafos de execução, permitindo que diferentes etapas de raciocínio sejam organizadas e coordenadas de forma estruturada.
Durante o projeto, foi construído um sistema multiagente para geração e revisão de redações, onde diferentes agentes são responsáveis por tarefas específicas como planejamento, pesquisa, geração de texto e reflexão crítica. Esses agentes são orquestrados por meio do LangGraph, que permite estruturar o fluxo de execução em forma de grafo, controlando estados, transições e ciclos de revisão.


## 🧠 Conceitos Fundamentais Praticados
Durante o desenvolvimento deste projeto, foram aplicados conceitos importantes de IA baseada em agentes e orquestração de fluxos inteligentes, incluindo:
- Criação de arquiteturas multiagentes com LangGraph
- Modelagem de estado do agente utilizando TypedDict
- Construção de grafos de execução com StateGraph
- Orquestração de diferentes etapas do processo (planejamento, pesquisa, geração e revisão)
- Implementação de loops de revisão automática controlados por condições no grafo
- Integração de LLMs (Gemini) para geração de texto
- Uso de structured outputs com Pydantic para geração de consultas de pesquisa
- Integração com Tavily para busca de informações externas
- Implementação de checkpoints com SQLite para persistência do estado do agente
- Criação de interfaces interativas com Gradio
- Streaming da execução do agente para acompanhar cada etapa do processo
- Construção de pipelines de agentes capazes de planejar, pesquisar, escrever e revisar conteúdo automaticamente
